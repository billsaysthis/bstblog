This list is more for my bookmark-like use but feel free to click away yourself on this list of C#-related material:

  • WindowsForms - an official MS site

  • Sells Brothers - Chris Sells now works for MS but has code and demo goodies and good articles, a well-populated blog, and more.
  • ASPxtreme ADO.NET Web Workshop
  • CodeSmith is a FREEWARE template based code generator.
  • CSharpFriends - self-proclaimed world's greatest C# community
  • Designing Resizable Windows Forms
  • RSS.NET - open source library for working with RSS
  • Shawn Wildermuth - he wrote the book I picked up today, Pragmatic ADO.NET. Hat tip to Chris Sells for the recommendation.
  • C# Scheduler - Code Project article on writing scheduling services.
  • Tim Dawson - some interesting looking, worthy of investigation Windows Controls: no charge for you to use in your products royalty-free!

Have fun if you find this stuff fun.
